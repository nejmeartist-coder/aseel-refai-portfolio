1. Program Overview & Goals

This document details the instructional design architecture for the 14-day global logistics and supply chain onboarding matrix. The curriculum shifts new hires from passive observers to autonomous operators through structured progressive disclosure.

Methodology: Cathy Moore’s Action Mapping & ADDIE Framework

Primary Tooling: Figma (UI Prototyping & Flow Mapping), Articulate Storyline

2. Core Learning Objectives (Performance-Based)

  • Given an active fulfillment exception flag, the operator will correctly diagnose inventory routing errors with 100% adherence to safety protocol.
  • Given cross-border customs manifests, the clerk will input tracking nodes into the warehouse management system (WMS) without downstream data entry discrepancies.
  • Given multi-tier tracking histories, the support agent will resolve shipment tracking inquiries within standard SLA timeframes.

3. Module Breakdown & Sequencing

Phase 1: System Navigation & Status Indicator Literacy (Days 1–4)

Focuses on visual literacy, interface navigation, and recognizing critical warehouse status alerts through simulated software dashboards.

Phase 2: Exception Handling & Branching Scenarios (Days 5–10)

Immersive branching scenarios where learners experience safe-failure branching workspaces, receiving real-time corrective feedback on inventory routing choices.

Phase 3: Autonomous Workflow Certification (Days 11–14)

Final practical performance check measuring task completion velocity and error-free fulfillment tracking.

4. Assessment & Evaluation Strategy

Evaluation is measured using Kirkpatrick’s Model:

  • Level 1 (Reaction): Learner engagement scores and navigation utility ratings.
  • Level 2 (Learning): Formative branching scenario decision accuracy and summative simulation tests.
  • Level 3 (Behavior): On-the-job error reduction tracked via fulfillment dashboard metrics.
